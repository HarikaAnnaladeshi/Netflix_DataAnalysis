# Netflix Movies Data Analysis

This project presents an end-to-end Exploratory Data Analysis (EDA) of a Netflix Movies dataset using Python. The analysis includes data cleaning, preprocessing, feature engineering, and visualization to uncover trends in movie popularity, genres, ratings, and release years.

## Objectives

* Clean and preprocess raw movie data.
* Handle missing values and duplicate records.
* Perform exploratory data analysis (EDA).
* Visualize key insights using Python libraries.
* Answer business-oriented questions through data analysis.

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Workflow

* Data Loading
* Data Inspection
* Data Cleaning
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Data Visualization
* Insights & Conclusion

## Key Insights

* Identified the most popular movie genres.
* Analyzed movie popularity and audience ratings.
* Explored release year trends.
* Examined relationships between popularity, vote count, and ratings.
* Generated visual insights to support data-driven conclusions.

This project demonstrates practical data analysis skills and serves as a portfolio project for aspiring Data Analysts.
